Fazer um git clone do projeto myrese no GitHub (clonei como 'myrese').

Seguir tutorial: https://www.sitepoint.com/deploy-your-rails-to-openshift/
(*) Criei app como 'myrese' usando rhc
(*) Renomeei 'myrese' para 'myrese-openshift'
(*) O remoto origin j√° veio do GitHub, o outro remoto openshift foi configurado manualmente

O ASSET NORMALMENTE NAO FICA COMO PRODUCTION NO OPENSHIFT, SOMENTE DEVELOPMENT: rhc set-env RAILS_ENV="production" -a myrese

SE DER PROBLEMA (COMO DEU UMA VEZ...): rhc unset-env RAILS_ENV -a myrese

EM ALGUM MOMENTO EXECUTEI: bundle exec rake assets:precompile

TIVE QUE EDITAR config/environments/production.rb
  config.assets.compile = true #false
  config.serve_static_files = true #ENV['RAILS_SERVE_STATIC_FILES'].present?
